如何获取环境ID？
1. 打开 tcb.cloud.tencent.com，微信扫码登录
2. 点击「新建环境」→ 选择「按量计费」→ 填写名称 → 创建
3. 进入「数据库」，依次创建集合：projects、plans、review_points
4. 每个集合「权限设置」→「所有用户可读写」
5. 控制台顶部括号里的内容即为环境ID，例如：myapp-1g3j4k5l

如何获取 Personal Access Token？
1. 登录 GitHub，点击右上角头像，选择 Settings
2. 左侧底部选择 Developer settings -> Personal access tokens -> Tokens (classic)
3. 点击 Generate new token (classic)，填写 Note，勾选 repo 权限
4. 点击 Generate token，立即复制保存（关闭页面后无法再查看）
5. 在目标仓库中确认已上传数据文件（或系统将自动创建）
