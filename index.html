如何获取环境ID？
1. 打开 tcb.cloud.tencent.com，微信扫码登录
2. 点击「新建环境」→ 选择「按量计费」→ 填写名称 → 创建
3. 进入「数据库」，依次创建集合：sync_state、sync_online、projects、plans
4. 每个集合「权限设置」→「所有用户可读写」
5. 控制台顶部括号里的内容即为环境ID，例如：myapp-1g3j4k5l
发布为在线网站
推荐方案：腾讯云静态托管（免费额度充足）
将本系统部署到腾讯云静态网站托管后，所有人通过浏览器即可访问，无需安装软件，配合云数据库实现真正的多人实时协作。
方案一：腾讯云静态托管（推荐）
1. 登录 tcb.cloud.tencent.com，进入云开发环境
2. 左侧菜单选「静态网站托管」→「文件管理」
3. 上传 index.html 文件（点"发布共享数据"先下载最新版）
4. 自动获得访问域名，如 https://xxx.tcloudbaseapp.com
5. 可在「域名信息」绑定自定义域名（可选）
方案二：GitHub Pages（完全免费）
1. 注册 github.com 账号
2. 新建仓库，上传 index.html
3. 仓库设置 → Pages → Source 选 main 分支 → 保存
4. 自动获得 https://用户名.github.io/仓库名/ 访问地址
方案三：Vercel/Netlify（简单快速）
1. 访问 vercel.com 或 netlify.com
2. 拖拽 index.html 文件到页面即可部署
3. 自动生成访问链接，全球 CDN 加速
打开腾讯云开发控制台
